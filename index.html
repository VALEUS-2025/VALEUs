Civil Society EU Engagement Visualisations

This interactive database visualizes how civil society organisations in Europe and beyond counteract contestations to EU values — freedom, democracy, equality, rule of law, and human rights — across domains such as peacebuilding, fundamental rights, sustainability, migration, trade, digitalization, conflict, and climate change.

Flow Map

Geographical Spread and Action Domains

Overview of organisations, their action domains, and links to their websites; filter by action domains.

The flow map provides an overview of the geographical spread of the civil society organisations, their action domains, and a link to their website. The map can be filtered by selecting action domains which are distinguished from each other through their colour schemes.

Sankey Diagram

Flows of Engagement Across Actors

Visualises connections between organisations, action domains, and addressees; wider flows indicate stronger ties.

This diagram illustrates the connections between different organizations, Action Domains, and Addressees through the flow of engagement. Each node represents an actor or category, while the arrows linking them are drawn with widths proportional to their value. These values originate from the underlying dataset: whenever an organization is connected to a domain, the connection is counted or weighted. If multiple records exist for the same link, their values are summed, meaning that thicker arrows represent stronger or more frequent ties. The algorithm then scales these values into pixel widths, so that the diagram fits neatly within the page while preserving the proportions of the flows. This makes it possible to see at a glance where engagement is concentrated, which pathways dominate, and how activity branches or converges across the network. The Sankey diagram emphasizes the magnitude of relationships and the distribution of connections across stages.

Interactive Network

Network Structure and Collaboration Clusters

ForceAtlas-based network of projects, organisations, addressees, and domains; reveals hubs, clusters, and bridges.

This diagram utilizes the ForceAtlas layout algorithm to visualize the connections between projects, action domains, addressees, and organizations in the VALEUs dataset. It highlights clusters and communities. ForceAtlas simulates a physical system where edges act like springs pulling related nodes closer, while nodes themselves repel one another. As a result, groups of highly connected actors cluster together, while less connected ones are pushed outward. Larger, more central nodes represent key hubs — projects or organizations with many ties — while smaller nodes at the edges reflect actors with fewer connections. The clusters, displayed in different colors, represent communities of engagement. Country → Light blue, Addressee → yellow, Organization → Purple, Action Domain → Magenta.
